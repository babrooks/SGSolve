Detailed Description

Model for stateCombo.

Model that interfaces between the state QComboBox and the underlying SGSolution object. This model automatically synchronizes the number and labels of the states with the SGSolution, and it indicates with an asterisk which state generated the best test direction for the current iteration. When a new solution is loaded, or when the current iteration is changed in the SGPlotController, the changeLayout slot is triggered so that the stateCombo is updated to reflect the true number of states and the identity of the best state.
