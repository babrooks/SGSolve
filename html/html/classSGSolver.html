Detailed Description

Class for solving stochastic games.

This class contains parameters for the algorithm, the solve method, as well as the data structure produced by solve. It calculates the equilibrium payoff correspondence corresponding to an SGGame object.

Examples:
abreusannikov.cpp, as_twostate.cpp, and kocherlakota.cpp.

void SGSolver::solve ( )

Solve routine.

Initializes a new SGApproximation object and iteratively generates it until one of the stopping criteria have been met. Stores progress in the data member.
